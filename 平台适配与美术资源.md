# 第五阶段：平台适配与美术资源替换

> 本文档为"三卡二"项目第五阶段开发任务清单。
> 目标：替换正式美术资源，适配微信小游戏/H5/原生 App 多端，完成可发布版本。
> 依赖：第四阶段（AI 机器人）已完成。

### 一、美术资源集成

- [ ] **1.1** 下载并集成扑克牌面资源
  - 54 张牌面 PNG（Kenney Playing Cards 或 Vector Playing Cards）

  - 花色图标（♠♥♦♣）
  - 整理到 `client/assets/resources/cards/` 并打包为 SpriteSheet
  - 📎 参考：[三卡二小程序.md - §14.2 扑克牌面方案](../三卡二小程序.md#142-方案一扑克牌面核心资源)

- [ ] **1.2** 下载并集成 UI 资源
  - 按钮素材（Kenney UI Pack：出牌/不出/确认/取消等）
  - 面板底图（叫分/扣底/叫搭档/结算面板）
  - 游戏图标（金币/时钟/皇冠/机器人/齿轮等）
  - 📎 参考：[三卡二小程序.md - §14.4 UI 按钮 & 面板 + §14.5 图标](../三卡二小程序.md#144-方案三ui-按钮--面板)

  - 牌桌背景（深绿色毡布纹理）
  - 大厅背景（深色木纹）
  - 字体：思源黑体 + Roboto Bold（数字位图字体）
  - 📎 参考：[三卡二小程序.md - §14.3 牌桌背景 + §14.6 字体](../三卡二小程序.md#143-方案二牌桌--大厅背景)

  - 结算：胜负大字 + 搭档揭晓 + 金币变化动画 + 再来一局
  - 📎 参考：[三卡二小程序.md - §7.2 首页大厅 + §7.5 结算](../三卡二小程序.md#72-启动页--首页大厅)
