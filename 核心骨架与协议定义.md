## 验收标准

| # | 验收项 | 通过条件 |
|---|--------|----------|
| V1 | 目录结构完整 | `sankaer/` 下 client/server/proto/scripts/deploy 目录齐全 |
| V2 | Proto 可编译 | 运行 `proto-gen.sh` 能成功生成 .pb.go 和 .ts 文件 |
| V3 | Go 服务可启动 | 每个 `cmd/*/main.go` 可 `go build` 通过 |
| V4 | 本地环境可启动 | `docker-compose up` 可拉起 Redis + MySQL + NATS |
| V5 | 客户端框架可编译 | TypeScript 代码无语法错误 |
